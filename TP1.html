TRABAJO PRÁCTICO Nº 1: Instalación y Características de los Sistemas Operativos

Objetivos de aprendizaje
  • Conocer las características básicas de diferentes SO (windows, linux).
  • Adquirir habilidad en la instalación y configuración de SO.
Unidad temática que incluye este trabajo práctico

Este trabajo práctico corresponde a la unidad Nº: 1 de la programación de la asignatura.

Consignas a desarrollar en el trabajo práctico

Sugerencia: muchos libros están disponibles digitalmente a través de la plataforma e-libro.

En el área de la informática no sólo es necesario poseer conocimientos de programación, sino también conocer las diversas herramientas que posibilitan el desarrollo de esta tarea. Es por esto que la instalación y configuración de sistemas operativos es de suma importancia: sustenta el resto de las actividades.

El teórico puede ser consultado en el libro “Instalación y configuración de sistemas operativos” de Enrique Bellido Quintero, Unidad 1, Tema 5.

Preguntas
  1. ¿Qué es un SO (Sistema Operativo)? De ejemplos de SO.

    2. Es un programa que administra el Hardware de una computadora. Proporciona las bases para los programas de aplicación y actúa como un intermediario entre el usuario y el hardware.

    Tareas que realiza un SO:

    • Realizar el interfaz Sistema-Usuario
    • Compartir los recursos de Hardware entre los usuaris
    • Permitir a los usuarios comaprtir sus datos entre ellos
    • Prevenir que las actividades de un usuario no interfieran en las de los demás usuarios
    • Calendarizar los recursos de los usuarios
    • Facilitar el acceso a los dispositivos de E/S
    • Recuperarse de fallas o errores
    • Llevar el control sobre el uso de los recursos

    Ejemplos de SO para PC:

    • Solaris
    • Linux
    • Windows 7
    • Mac OS X
    • FreeBSD

    Ejemplos de SO para Móviles:

    • Android
    • Windows phone
    • iOS
    • Blackberry
  2. ¿Qué diferencia existe entre los requisitos mínimos y los recomendados de hardware especificados por los SO?

    3. Con la configuración del hardware mínima el rendimiento del sistema suele ser pobre, ya que el SO sólo podrá ejecutar funciones básicas.

    Con la configuración de hardware recomendada el SO funciona a una velocidad adecuada.

  3. ¿Qué es la BIOS? ¿Qué función cumple?

    4. Significado de siglas: BASIC INPUT/OUTPUT SYSTEM

    La BIOS es el software que localiza y reconoce los dispositivos necesarios para cargar el Sistema Operativo en la memoria RAM.

    Es el encargado de gestionar el hardware y permite gestionar el arranque, es decir, seleccionar desde qué dispositivo debería leer el SO.

    También Puede controlar la frecuencia de la RAM, de la tarjeta gráfica, las revoluciones de los ventiladores, periféricos, etc.

    Está integrado en un chip en la placa madre, y el firmware puede ser actualizado y grabado en la memoria flash que posee integrada.

    Sólo puede ser modificado por una actualización del fabricante. Su memoria es de solo lectura.

    Una vez que el BIOS comprueba que los componentes de la computadora funcionan correctamente, le pasa el control del hardware al sistema operativo.

  4. ¿Qué sucede al formatear un disco duro? ¿Qué es el formato rápido?

    5. Formatear un disco duro consiste en restablecer a este a su estado original para ser reutilizado o reescrito con una nueva información.

    Al realizar esta operación se elimina toda la información contenida en el.

    El formato rápido no elimina físicamente los archivos, no reconstruye el sistema de archivos eliminando todos los datos en el proceso, ni comprueba los sectores defectuosos, sólo construye una nueva tabla sobre la anterior, ocultando los datos que estaban almacenados.

  5. ¿Qué es una partición? ¿Qué ventajas tiene particionar un disco duro? ¿Qué son las particiones primarias y secundarias?

    6. Una partición es la división de espacio que se le asigna a un disco duro.

    Un disco duro puede tener varias particiones, que son de cierto modo independientes entre si, dado que cada una de ellas tiene su propio sistema de archivos o formato, y el SO lo reconoce como un disco independiente (aunque físicamente sea uno).

    Algunas ventajas de particionar un disco duro:

    Se puede instalar el SO en una partición primaria y usar otras particiones secundarias para almacenamiento de datos.

    Se puede formatear una partición sin necesidad de formatear todo el disco.

    Los datos que se introducen en cada una de ellas no afectan al espacio de las otras. Y si borramos o formateamos una ellas, el resto no sufrirá cambios.

    Particiones Primarias y Secundarias

    • Partición Primaria: Es la primera y la partición más importante, ya que se encarga de iniciar la carga del Sistema Operativo. Es indispensable para almacenar y arrancar el Sistema Operativo.
    • Partición Extendida o Secundaria: Son particiones con el único objetivo de almacenar información. Este tipo de partición no es booteable, no permite arrancar cargando el Sistema Operativo.
  6. ¿Qué diferencia existe entre la instalación mínima, típica o estándar y personalizada de un programa? Explique ventajas y desventajas.
    • Instalación Mínima: es la instalación con la menor cantidad de funcionalidades o características que necesita un programa para funcionar;
    • Instlación Típica o Estánadar: es la instalación que suele ser “recomendada” por el fabricante, ya que instala lo mínimo para funcionar, pero agrega características y funcionalidades que pueden mejorar el uso o el rendimiento del mismo
    • Instalación Personalizada: es la que permite al usuario elegir que componentes del programa instalar.

    Cuadro comparativo:

    Tipo de Instalación Ventaja Desventaja
    Mínima Es recomendada para computadoras de bajos recursos y poco espacio en disco. Suele ser carente de funcionalidades extras, que puede ofrecer el programa. Suele pedir el CD/DVD de instalación cuando no encuentre alguna librería faltante.
    Típica Tener la seguridad que todas las librerías estan disponibles en cualquier comento. Opcupa mas espacio en disco, se suele instalar programas y plugins asociados que en la práctica no son usados.
    Personalizada Te da la opción de instalar lo necesario, evitando programas innecesarios. No es aconsejable este tipo de instalación para ususarios novatos.
  7. ¿Qué es una máquina virtual? ¿Qué ventajas posee?

    8. Es un software que simula a un computadora y puede ejecutar programas.

    Se puede configurar para que los procesos que ejecutan están limitados por los recursos proporcionados.

    Habitualmente se usa para ejecutar sistemas operativos y "probarlos".

    Características:

    • La MV no puede acceder al resto de datos de la máquina anfitrion.
    • Para funcionar, una MV mapea los dispositivos virtuales que ofrece a su invitado con los dispositivos reales presentes en la máquina física.
    • Copias de Seguridad y Clonación
    • Crear entornos de prueba

    Algunas aplicaciones para gestionar MV:

    • VMWare
    • Virtual Box
    • Qemu
    • Parallels

    Algunas Ventajas:

    • Probar sistemas operativos
    • Ejecutar programas antiguos
    • Usar software que no esta disponible en nuestro SO
    • Probar una aplicación en distintos sistemas
    • Copias de Seguridad y Clonación
    • Crear entornos de prueba
    • En la industria de telecomuniaciones se esta migrando todos los servidores físicos a servidores virtuales montados sobre datacenter, solucionando un problema típico de reposición de HW debido al gran uso de recursos de estos servidores.
